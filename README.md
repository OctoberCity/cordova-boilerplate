# common-app

## Project setup
```
npm install && cd cordova && mkdir www && cordova platform add (android/ios/etc)
```

### Compiles and hot-reloads for development android
```
npm run devand
```

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files
```
npm run lint
```
